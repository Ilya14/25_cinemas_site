API

Сервис «Что в кино» имеет API в формате JSON. API доступен по адресу https://whatatcinema.herokuapp.com/api. Структура данных приведена ниже: 

      [
        {
          'name': ...,
          'rating': ...,
          'rating_count': ...,
          'cinemas_count': ...,
          'film_synopsys': ...,
          'film_img': ...,
          'movie_afisha_url': ...
        },
        ...

Описание предоставляемых данных приведено в таблице ниже.
JSON-параметр Описание
name Название фильма
rating Рейтинг фильма по данным Кинопоиск
rating_count Количество оценок по данным Кинопоиск
cinemas_count Количество кинотеатров, в которых идет фильм, по данным Афиша
film_synopsys Описание фильма
film_img Ссылка на постер к фильму
movie_afisha_url Ссылка на страницу фильма на Афиша
